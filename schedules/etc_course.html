1. 중요도 순서 (/는 동시에, ()는 끝나고나서)

  1. 선형대수(딥러닝), 확률론(통계학), ML사이언스(강화학습)
  2. 보안(해킹실습->리버싱/Win32&MFC), 양자컴퓨팅/물리학, ML엔지니어링
  3. 자연어처리(DL NLP), 컴퓨터비전(DL Vision), 클라우드/DevOps
  4. 암호학/정수론, 웹/앱개발, 미적분학/수치해석
  5. 쓰레드(가비지컬렉팅), 시스템프로그래밍, 소켓프로그래밍
  6. C언어, 어셈블리어, 분산처리/Spark, 가상화
  7. 그래픽스, 정보검색, 블록체인, 두뇌-컴퓨터인터페이스

2. 공부 자료

  • 선형대수학 : 3B1B => 티스토리블로그
  • 딥러닝 : 이안굿펠로우책보기
  • 확률론 : 이상준교수님(경희대), 이승철교수님(연세대) 번갈아서
  • 통계학 : 여인권교수님(숙명여대) 통계학 1, 통계학2듣기
  • ML사이언스 : 머피책, 비숍책 같이 보면서 공부하고 정리하기
  • 강화학습 : 김성훈교수님(홍콩과기대) => 유튜브(팡요랩)이론 + 이승철교수님(연세대)구현
  • 보안개론 : 홍성혁교수님(백석대) 보안개론/해킹강의
  • 해킹실습 : 유튜브(나동빈) 시스템해킹, 웹해킹, 게임해킹, ARP스푸핑 따라해보기
  • 리버싱 : 리버싱핵심원리 책보기 + 유튜브(스무디TV) => 유튜브(나우캠퍼스)
  • 양자컴퓨팅 : 양자 프로그래밍, 양자정보학강의(이해웅교수님) 책보기
  • ML 엔지니어링 : 송호연님 로드맵에 있는 것들 공부 (서빙/엔지니어링 관점, JIT, Horovod 등등)
  • 자연어처리 : 스탠포드대학, 조경현교수님(뉴욕대학교)
  • 컴퓨터비전 : 황성수교수님(한동대), 유튜브(멈춤보단천천히라도)
  • 클라우드 : 오픈스택 강의보면서 클라우드 시스템 만들어보기
  • DevOps : 도커, 쿠버네티스, 쿠브플로우, AWS, Azure, Jenkins 등등 배워보기
  • 정수론 : 유튜브(쑤튜브)
  • 암호학 : 조인준교수님(배재대) =>홍득조교수님(전북대)
  • 웹개발 : 유튜브(노마드코더) => NodeJS, ReactJS, 유튜브(불스), 마이바티스/하이버네이트등 공부
  • 앱개발 : 플러터 강의, 안드로이드 Okhttp/레트로핏/ 등 유명 라이브러리 공부하기
  • 미적분 : 3B1B 보기 => 이상준교수님 미적분학1, 미적분학2
  • 수치해석 : 유튜브(내가이해한기계공학)
  • 쓰레드 : 깃헙블로그
  • 소켓프로그래밍 : 유튜브(달콤)보기 => 뇌를자극하는 TCP/IP
  • 시스템프로그래밍 : 유닉스 시스템프로그래밍(고려대) + 윈도우즈 시스템프로그래밍
  • C언어 : 유튜브(두들낙서) C언어 에서 모르는 내용 골라보기
  • 어셈블리 : 유튜브(스무디TV)
  • 분산처리(빅데이터) : 박영택교수님(숭실대)
  • 정보검색론 : 박세영교수님(경북대)
  • 블록체인 : 프린스턴대학
  • 두뇌-컴퓨터 인터페이스 : 캘리포니아대학
  • 그래픽스 : 홍정모교수님, 한정현교수님, 유튜브
